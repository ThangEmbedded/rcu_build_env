# Cấu trúc thư mục dự án

- app/: code chính
- platform/: code nền tảng (STM32, ESP32, ...)
- components/: các thành phần mở rộng (network, protocols, ...)
- drivers/: driver phần cứng (gpio, uart, flash, ...)
- middleware/: phần mềm trung gian (FreeRTOS, ...)
- services/: dịch vụ
- tests/: kiểm thử
- docs/: tài liệu
